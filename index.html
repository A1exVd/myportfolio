Мой стэк

Node.js — это кроссплатформенная среда выполнения JavaScript с открытым исходным кодом, построенная на движке Chrome V8, которая позволяет запускать код JavaScript вне браузера, в основном на сервере.

TypeScript (TS) — это язык программирования, который расширяет JavaScript с помощью статической типизации, что делает код более надежным и масштабируемым.

JavaScript (JS) — это язык программирования, создающий интерактивные элементы на веб-сайтах, такие как анимации, формы, игры и динамический контент, благодаря чему делает их более динамичными и удобными для пользователя.

CSS (Cascading Style Sheets — каскадные таблицы стилей) — это язык, который используется для описания внешнего вида документов, написанных на HTML или XML.

HTML (HyperText Markup Language — «язык гипертекстовой разметки») — это стандартизированный язык, используемый для создания и структурирования содержимого веб-страниц.

Python — это высокоуровневый, универсальный язык программирования общего назначения, известный своим простым и читаемым синтаксисом.

Java — это кроссплатформенный, объектно-ориентированный язык программирования общего назначения, который работает на миллиардах устройств по всему миру.
